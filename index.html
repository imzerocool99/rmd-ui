❓ Anticipated Questions & Answers
Is this using real money / real trades?
This POC uses Alpaca Paper Trading — a simulated environment. No real orders are placed. Production upgrade is straightforward: replace the paper trading API key with a live one.
Where does the AI run — is client data going to the cloud?
The AI (Ollama/Phi-3) runs 100% locally on the machine. No client data leaves the server. The language model is downloaded once and runs offline.
Can it handle multiple clients at once?
Yes — the agent is stateless per request. It can process multiple clients in parallel. The scheduler runs automatically for all eligible accounts.
What about regulatory compliance and audit trail?
Every agent decision is logged with timestamps in the Logs tab. The explanation of why each asset was selected is stored per client. Full audit trail is available for regulatory review.
How accurate is the tax savings calculation?
The POC uses a 24% marginal tax rate as a baseline. In production, the actual client tax bracket and state taxes would be integrated from the firm's tax engine for precise calculations.
